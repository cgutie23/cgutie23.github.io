Website Evaluations

Uncut

This website is a site that is used to showcase tons of free fonts that the user may use for their personal uses. It displays them throughout the website with nice monotone colors and an easy to read layout.

  • C: The Contrast of this site is extremely good, easy on the eyes, and easy to read.
  • R: The Repetition of this site is very consistent, which gives it an easy to follow layout.
  • A: The Alignment of this site is very consistent, allowing for a smooth and consistent experience throughout the site.
  • P: The Proximity of this site is very uniform and well spaced. This makes the site extremely easy to read and understand which text corresponds to which title/section.

Lings Cars

This website is for selling cars and has all different types of payment plants etc. It also has a lot of different types of cars.

  • C: The Contrast of this site is very bad due to the color choices. These color choices are very bright and not easy on the eyes of the viewer.
  • R: The Repetition of this site is not so bad. The product offerings all share the same styling.
  • A: The Alignment of this site is also not bad, as each section of the website is consistent in terms of alignment, making it easy to follow.
  • P: The Proximity of this site is not bad. Each description is in its corresponding section, making it easy to follow.
